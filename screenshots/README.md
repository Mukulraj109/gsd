## Screenshot Guidelines

- **Resolution:** 1920x1080 or 1440x900 (desktop view)
- **Format:** PNG with transparency where applicable
- **Browser:** Chrome or Firefox with clean UI
- **Zoom:** 100% or 90% for better card visibility
- **State:** 
  - Logged in for app pages
  - Show hover states for interactive elements
  - Ensure all text is readable

## Quick Capture Commands

If using a screenshot tool:

```bash
# Full page screenshots
# 1. Navigate to each URL
# 2. Open dev tools (F12)
# 3. Cmd/Ctrl + Shift + P
# 4. Type "screenshot" and select "Capture full size screenshot"
```

## Alternative: Use Placeholder Images

If screenshots aren't available yet, you can:
1. Use placeholder images from https://placehold.co/
2. Add them with: `placehold.co/1920x1080/0039A6/FFFFFF/png?text=Dashboard`
3. Replace with real screenshots later
